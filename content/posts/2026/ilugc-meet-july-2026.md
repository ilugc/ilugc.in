### Talk 0

**Topic:** Copy Fail vulnerability \
**Description:** Copy Fail is a Linux kernel vulnerability that was disclosed 
few weeks back. It allows a straightforward privilege escalation and
affects many distributions. In this talk we will go over the root cause of
the vulnerability and understand how the attack works. If time permits,
we'll discuss the other recent vulnerabilities and how to protect \
against them.
