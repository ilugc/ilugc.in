### Talk 0

**Topic**: Talos: A modern OS for Kubernetes\
**Description**: Talos is a secure, immutable and minimal operating system for Kubernetes. It uses a declarative configuration for fast deployments and can be API managed. It removes configuration drift by treating infrastructure as code. It is free and open source software, production-ready and supported by Sidero Labs. In this session, we will learn and explore the salient features of Talos.\
**Duration**: 45 minutes\
**Full Name**: Shakthi Kannan\
